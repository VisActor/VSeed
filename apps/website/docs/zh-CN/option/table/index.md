# Table

:::info{title=推荐}
\- 推荐字段配置: `任意`个指标, `任意`个维度

\- 支持数据重塑: 至少`任意`个指标, `任意`个维度

:::

:::note{title=描述}
表格，适用于详细数据展示场景，行列分明，便于查看具体数值

适用场景:

\- 需要展示详细数据明细

\- 数据项需要精确比对

\- 展示多维度数据属性

:::

:::info{title=编码映射}
仅支持配置维度树与指标树, 默认encoding到column

:::

:::warning{title=Warning}
数据要求:

\- 至少1个维度字段

\- 至少1个度量字段

\- 维度字段会作为表格的列标题

默认开启的功能:

\- 默认开启排序、筛选、分页功能

:::
