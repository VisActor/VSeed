# dimensions

**Type:** `Dimensions | undefined`

:::note{title=描述}
维度, 第一个维度被映射到Y轴, 其余维度会与指标名称(存在多个指标时)合并, 作为图例项展示.

:::

**示例**
[{id: "date", alias: "日期"}, {id: "value", alias: "数值"}]




## id

**Type:** `string`

## alias

**Type:** `string | undefined`

## location

**Type:** `"dimension" | "rowDimension" | "columnDimension" | undefined`
