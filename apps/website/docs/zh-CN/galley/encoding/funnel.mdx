# funnel

## 介绍
:::info
漏斗图是一种特殊的图表，用于展示数据在不同阶段的转化或流失情况。它通常用于分析销售流程、网站访问流程等，通过梯形的宽度或高度来表示每个阶段的数量，从而直观地展示转化率。
:::

## Encoding

漏斗图支持以下视觉通道:

- `color`: **必选**，颜色映射通道, 支持放入多个维度或1个指标。用于区分不同的阶段。
- `detail`: **可选**，细分映射通道, 支持放入多个维度。当需要在同一个颜色系列下展示更细粒度的数据时使用。
- `label`: **可选**，标签映射通道, 支持放入多个维度或多个指标。用于在图表上直接显示数据值。
- `tooltip`: **可选**，提示映射通道, 支持放入多个维度和多个指标。用于在鼠标悬停时显示详细信息。
