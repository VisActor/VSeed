# roseParallel

## 介绍
:::info
分组玫瑰图是玫瑰图的一种变体，它在极坐标系下通过扇形的半径和角度来表示数据，并允许将数据分组显示。这使得在同一图表中比较不同组别的数据分布和趋势变得更加直观。
:::

## Encoding

:::tip
在分组玫瑰图中, **measures** 中的所有指标会自动合并，并映射到扇形的 **半径** 上，用于表示扇形的大小。
:::

分组玫瑰图支持以下视觉通道:

- `angle`: **可选**，映射到角度的字段, 支持放入多个维度。通常表示类别或时间。
- `color`: **可选**，颜色映射通道, 支持放入多个维度或1个指标。用于区分不同的数据系列。
- `detail`: **可选**，详情映射通道, 支持放入多个维度。当需要在同一个颜色系列下展示更细粒度的数据时使用。
- `label`: **可选**，标签映射通道, 支持放入多个维度或多个指标。用于在图表上直接显示数据值。
- `tooltip`: **可选**，提示映射通道, 支持放入多个维度和多个指标。用于在鼠标悬停时显示详细信息。
