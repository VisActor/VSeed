# 编码映射

## 什么是编码映射

编码映射是一种将数据属性映射到视觉通道的机制，是数据可视化中的核心概念。通过编码映射，我们可以将抽象的数据转化为直观的视觉元素，如颜色、位置、大小等，从而帮助用户快速理解数据的分布、趋势和关系。

在 VSeed 中，每种图表类型都支持一系列特定的视觉通道，用于展示不同维度的数据信息。下面将详细介绍各种图表类型及其支持的编码映射通道。

## column

**介绍**
:::info
柱状图使用垂直的条形来显示不同类别之间的比较。它是最常见的图表类型之一，非常适合用于类别名称较短的情况，或者当您想要强调数值随时间的变化时。X轴代表类别，Y轴代表数值。
:::

**Encoding**

:::tip
在柱状图中, **measures** 中的所有指标会自动合并，并映射到 **Y轴** 上，用于表示柱子的高度。
:::

柱状图支持以下视觉通道:

- `x`: **可选**，映射到X轴的字段, 支持放入多个维度。通常表示类别或时间。
- `color`: **可选**，颜色映射通道, 支持放入多个维度或1个指标。用于区分不同的数据系列。
- `detail`: **可选**，细分映射通道, 支持放入多个维度。当需要在同一个颜色系列下展示更细粒度的数据时使用。
- `label`: **可选**，标签映射通道, 支持放入多个维度或多个指标。用于在图表上直接显示数据值。
- `tooltip`: **可选**，提示映射通道, 支持放入多个维度和多个指标。用于在鼠标悬停时显示详细信息。

**示例**

<EncodingColumn />

```json
{
  "chartType": "column",
  "measures": [
    {
      "id": "sales",
      "alias": "Sales",
      "autoFormat": true

      "id": "discount",

    }

  "dimensions": [
    { "id": "year", "alias": "Yearn" },

  "encoding": {
```

## columnPercent

**介绍**
:::info

:::

- `label`: **可选**，标签映射通道, 支持放入多个维度或多个指标。用于在图表上直接显示数据值。
- `tooltip`: **可选**，提示映射通道, 支持放入多个维度和多个指标。用于在鼠标悬停时显示详细信息。

## heatmap

**介绍**
:::info
热力图通过颜色的变化来表示二维矩阵中数值的大小。它非常适合用于展示数据的分布、密度和强度，尤其是在处理大量数据时。颜色越深，通常表示数值越大。
:::

**Encoding**

:::tip
在热力图中, **measures** 中的所有指标会自动合并，并映射到 **颜色** 通道上，用于表示热力块的颜色深浅。
:::

热力图支持以下视觉通道:

- `x`: **可选**，x轴映射通道, 支持放入多个维度。
- `y`: **可选**，y轴映射通道, 支持放入多个维度。
- `color`: **可选**，颜色映射通道, 支持放入多个维度或1个指标。用于表示数值的大小。
- `label`: **可选**，标签映射通道, 支持放入多个维度或多个指标。用于在图表上直接显示数据值。
- `tooltip`: **可选**，提示映射通道, 支持放入多个维度和多个指标。用于在鼠标悬停时显示详细信息。

**示例**

## funnel

**介绍**
:::info
漏斗图是一种特殊的图表，用于展示数据在不同阶段的转化或流失情况。它通常用于分析销售流程、网站访问流程等，通过梯形的宽度或高度来表示每个阶段的数量，从而直观地展示转化率。
:::

**Encoding**

:::tip
在漏斗图中, **measures** 中的所有指标会自动合并，并映射到漏斗图的 **高度** 或 **宽度** 上，用于表示每个阶段的数量。
:::

漏斗图支持以下视觉通道:

- `color`: **可选**，颜色映射通道, 支持放入多个维度或1个指标。用于区分不同的阶段。
- `detail`: **可选**，细分映射通道, 支持放入多个维度。当需要在同一个颜色系列下展示更细粒度的数据时使用。
- `label`: **可选**，标签映射通道, 支持放入多个维度或多个指标。用于在图表上直接显示数据值。
- `tooltip`: **可选**，提示映射通道, 支持放入多个维度和多个指标。用于在鼠标悬停时显示详细信息。

**示例**

<EncodingFunnel />

```json
{
  "chartType": "funnel",
  "measures": [
    {
      "id": "sales",
      "alias": "Sales",
      "autoFormat": true
```

## dualAxis

**介绍**
:::info
双轴图用于在同一个图表中展示两个具有不同单位或量级的指标。它通过使用主、次两个Y轴，将两个相关的指标进行对比，非常适合分析两个变量之间的关系，例如销售额和增长率。
:::

**Encoding**

:::tip
在双轴图中, **measures** 可以配置两个指标组。第一个指标组的指标会自动合并，并映射到 **主Y轴** 上；第二个指标组的指标会自动合并，并映射到 **次Y轴** 上。
:::

双轴图支持以下视觉通道:

- `x`: **可选**，映射到X轴的字段, 支持放入多个维度。通常表示时间或类别。
- `color`: **可选**，颜色映射通道, 支持放入多个维度或1个指标。用于区分不同的数据系列。
- `detail`: **可选**，细分映射通道, 支持放入多个维度。当需要在同一个颜色系列下展示更细粒度的数据时使用。
- `label`: **可选**，标签映射通道, 支持放入多个维度或多个指标。用于在图表上直接显示数据值。
- `tooltip`: **可选**，提示映射通道, 支持放入多个维度和多个指标。用于在鼠标悬停时显示详细信息。

**示例**

<EncodingDualAxis />

```json
{
  "chartType": "dualAxis",
  "measures": [
    {
      "id": "sales",
      "alias": "Sales",
      "autoFormat": true
    },
    {
      "id": "profit",
      "alias": "Profit",
      "autoFormat": true
    },
    {
      "id": "discount",
      "alias": "Discount",
      "autoFormat": true
    }
  ],
  "dimensions": [
    { "id": "month", "alias": "Month" },
    { "id": "year", "alias": "Year" }
  ],
  "dualMeasures": [
    {
      "primaryMeasures": [{ "id": "profit" }, { "id": "sales" }],
      "secondaryMeasures": [{ "id": "discount" }]
    }
  ],
  "encoding": {
    "x": ["month"],
    "color": ["__MeaName__"],
    "detail": ["year"]
  },
  "dataset": [
    { "year": "2019", "month": "Jan", "profit": 10000, "sales": 200000, "discount": 0.5 },

    { "year": "2019", "month": "Apr", "profit": 40000, "sales": 800000, "discount": 0.2 },
```

## donut

**介绍**
:::info
环形图是一种特殊的饼图，中心有一个空白区域，通常用于显示总计或关键指标。它非常适合展示单一维度数据的占比关系，同时通过中心区域提供额外的信息。
:::

**Encoding**

:::tip
在环形图中, **measures** 中的所有指标会自动合并，并映射到扇形的 **半径** 上，用于表示扇形的大小。
:::

环形图支持以下视觉通道:

- `color`: **可选**，颜色映射通道, 支持放入多个维度或1个指标。用于区分不同的扇区。
- `detail`: **可选**，细分映射通道, 支持放入多个维度。当需要在同一个颜色系列下展示更细粒度的数据时使用。
- `label`: **可选**，标签映射通道, 支持放入多个维度或多个指标。用于在图表上直接显示数据值。
- `tooltip`: **可选**，提示映射通道, 支持放入多个维度和多个指标。用于在鼠标悬停时显示详细信息。
