# dimensions

**Type:** `Dimensions | undefined`

:::note{title=描述}
维度



饼图的所有维度会与指标名称(存在多个指标时)合并成一个维度, 映射到角度, 并作为图例项展示

:::

**示例**
[{id: 'category', alias: '类别'}]

**Type:** `string`

## alias

**Type:** `string | undefined`

## location

**Type:** `"dimension" | "rowDimension" | "columnDimension" | undefined`
