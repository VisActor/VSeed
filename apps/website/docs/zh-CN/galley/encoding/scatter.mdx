# scatter

## 介绍
:::info
散点图使用点来显示两个或多个变量之间的关系。每个点的位置由其在X轴和Y轴上的值决定。它非常适合用于发现数据中的模式、趋势和异常值，尤其是在分析两个连续变量之间的相关性时。
:::

## Encoding

:::tip
在散点图中, **measures** 可以配置两个指标组。第一个指标组的指标会自动合并，并映射到 **X轴** 上；第二个指标组的指标会自动合并，并映射到 **Y轴** 上。
:::

散点图支持以下视觉通道:

- `color`: **可选**，颜色映射通道, 支持放入多个维度或1个指标。用于区分不同的数据系列或表示第三个变量。
- `detail`: **可选**，详情映射通道, 支持放入多个维度。当需要在同一个颜色系列下展示更细粒度的数据时使用。
- `label`: **可选**，标签映射通道, 支持放入多个维度或多个指标。用于在图表上直接显示数据值。
- `tooltip`: **可选**，提示映射通道, 支持放入多个维度和多个指标。用于在鼠标悬停时显示详细信息。
