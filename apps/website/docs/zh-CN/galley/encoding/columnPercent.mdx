# columnPercent

## 介绍
:::info
百分比柱状图是柱状图的一种变体，它将每个类别的值显示为总量的百分比。这使得比较不同类别在整体中的相对贡献变得容易。Y轴的范围总是从0%到100%。
:::

## Encoding

:::tip
在百分比柱状图中, **measures** 中的所有指标会自动合并，并映射到 **Y轴** 上，用于表示柱子的高度。Y轴将以百分比的形式显示数据的占比关系。
:::

百分比柱状图支持以下视觉通道:

- `x`: **可选**，映射到X轴的字段, 支持放入多个维度。通常表示类别或时间。
- `color`: **可选**，颜色映射通道, 支持放入多个维度或1个指标。用于区分并列的各个数据系列。
- `detail`: **可选**，细分映射通道, 支持放入多个维度。当需要在同一个颜色系列下展示更细粒度的数据时使用。
- `label`: **可选**，标签映射通道, 支持放入多个维度或多个指标。用于在图表上直接显示数据值。
- `tooltip`: **可选**，提示映射通道, 支持放入多个维度和多个指标。用于在鼠标悬停时显示详细信息。
