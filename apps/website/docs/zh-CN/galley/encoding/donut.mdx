# donut

## 介绍
:::info
环形图是一种特殊的饼图，中心有一个空白区域，通常用于显示总计或关键指标。它非常适合展示单一维度数据的占比关系，同时通过中心区域提供额外的信息。
:::

## Encoding

环形图支持以下视觉通道:

- `color`: **可选**，颜色映射通道, 支持放入多个维度或1个指标。用于区分不同的扇区。
- `detail`: **可选**，细分映射通道, 支持放入多个维度。当需要在同一个颜色系列下展示更细粒度的数据时使用。
- `label`: **可选**，标签映射通道, 支持放入多个维度或多个指标。用于在图表上直接显示数据值。
- `tooltip`: **可选**，提示映射通道, 支持放入多个维度和多个指标。用于在鼠标悬停时显示详细信息。
