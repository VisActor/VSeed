# dimensions

**Type:** `Dimensions | undefined`

:::note{title=描述}
维度, 第一个维度会放至X轴, 其余维度会与指标名称(存在多个指标时)合并, 作为图例项展示.:::

**示例**
[{id: 'month', alias: '月份'}]
