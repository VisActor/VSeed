# dimensions

**Type:** `Dimensions | undefined`

**描述:**
维度
  
  折线图的第一个维度被映射到X轴, 其余维度会与指标名称(存在多个指标时)合并, 作为图例项展示

**示例:**
[{id: "month", alias: "月份"}]


## id

**Type:** `string`

## alias

**Type:** `string | undefined`

## location

**Type:** `"dimension" | "rowDimension" | "columnDimension" | undefined`
