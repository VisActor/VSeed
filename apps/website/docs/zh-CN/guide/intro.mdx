# 功能概览

| Feature   | Description                                           | Link                                 |
| ---------- | ---------------------------------------------- | ---------------------------------------- |
| 图表类型   | 支持多种图表类型的快速切换                     | [示例](./intro/chartTypeSwitch.mdx)      |
| 数据重塑   | 支持不同维度和指标的数据结构转换与重塑         | [示例](./intro/dataReshape.mdx)          |
| 透视与组合 | 指标组合与维度透视                             | [示例](./intro/pivotAndCombine.mdx)      |
| 数值格式   | 提供丰富的数值格式化选项                       | [示例](./intro/dataFormat.mdx)           |
| 分析功能   | 支持数据排序、图例排序功能                     | [示例](./intro/dataAnalysis.mdx)         |
| 条件样式   | 基于数据条件设置不同的图元样式                 | [示例](./intro/conditionalStyle.mdx)     |
| 数据标注   | 支持为图表添加数据点、水平线、垂直线和区域标注 | [示例](./intro/annotation.mdx)           |
| 跨平台渲染 | VSeed支持序列化，可在不同平台间传输和渲染      | [示例](./intro/crossPlatformRender.mdx)  |
| 分离式构建 | 支持数据处理与渲染分离，减轻客户端数据处理压力 | [示例](./intro/separateBuild.mdx)        |
| 国际化     | 支持多语言配置和展示                           | [示例](./intro/internationalization.mdx) |
