# heatmap

## 介绍
:::info
热力图通过颜色的变化来表示二维矩阵中数值的大小。它非常适合用于展示数据的分布、密度和强度，尤其是在处理大量数据时。颜色越深，通常表示数值越大。
:::

## Encoding

热力图支持以下视觉通道:

- `x`: **必选**，x轴映射通道, 支持放入多个维度。
- `y`: **必选**，y轴映射通道, 支持放入多个维度。
- `color`: **必选**，颜色映射通道, 支持放入多个维度或1个指标。用于表示数值的大小。
- `label`: **可选**，标签映射通道, 支持放入多个维度或多个指标。用于在图表上直接显示数据值。
- `tooltip`: **可选**，提示映射通道, 支持放入多个维度和多个指标。用于在鼠标悬停时显示详细信息。
