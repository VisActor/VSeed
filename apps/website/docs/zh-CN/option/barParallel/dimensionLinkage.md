# dimensionLinkage

**Type:** `DimensionLinkage | undefined`

:::note{title=描述}
当图表开启透视功能或者指标组合的是否，是否开启维度联动功能

当hover 到某个维度值时，联动高亮其他图表中相同维度值的数据

:::


## enable

**Type:** `false | true`

:::note{title=描述}
是否开启透视图表维度联动

:::

## showTooltip

**Type:** `boolean | undefined`

:::note{title=描述}
是否显示所有维度对应子图表的Tooltip提示信息

:::

## showLabel

**Type:** `boolean | undefined`

:::note{title=描述}
是否显示crosshair 对应的标签

:::
