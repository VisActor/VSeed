# area

## 介绍
:::info
面积图是一种用于可视化定量数据随时间或其他连续变量变化的图表。它通过在数据点之间填充颜色来强调数量的多少，非常适合展示趋势和比较不同组别的数据。
:::

## Encoding

:::tip
在面积图中, **measures** 中的所有指标会自动合并，并映射到 **Y轴** 上，用于表示面积的高度。
:::

面积图支持以下视觉通道:

- `x`: **可选**，映射到X轴的字段, 支持放入多个维度。通常表示时间或类别。
- `color`: **可选**，颜色映射通道, 支持放入多个维度或1个指标。用于区分不同的数据系列。
- `detail`: **可选**，细分映射通道, 支持放入多个维度。当需要在同一个颜色系列下展示更细粒度的数据时使用。
- `label`: **可选**，标签映射通道, 支持放入多个维度或多个指标。用于在图表上直接显示数据值。
- `tooltip`: **可选**，提示映射通道, 支持放入多个维度和多个指标。用于在鼠标悬停时显示详细信息。
